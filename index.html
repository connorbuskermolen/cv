Formula 1 Fan-Made Website

This ongoing project was started out of my love for Formula 1 as well as a way to build on my development skills. At the moment the website holds some general information about drivers and some statistcs on the sport, with more content continuously being added. I mainly used the React framework to devlop this website and used Saas for any styles and animations. You can visit my website at https://f1-pro.herokuapp.com/ and check it out for yourself! Any source code can be found at my Github page, the link is on the bottom of this site.

CISC 332 Website

This website was created for a school project in a Database Management course. The goal of the project was to create a simple website for an animal rescue tracker. The bulk of the project was creating the database starting with the schema. It then progressed forward to building the actual database and populating it with data. This was done through MySQL using the MariaDB software. Once the database was created, a simple front-end site was made using HTML and CSS to show off the project, connecting the site to the database with PHP.
